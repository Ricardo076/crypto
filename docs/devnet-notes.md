# Solana Devnet Notes

## Commands

```bash
solana config set --url devnet
solana config get
solana address
solana airdrop 2
solana balance
```

## Security checklist

- Do not commit seed phrases.
- Do not commit private key JSON files.
- Do not use main wallets for tests.
- Treat all public repos as fully visible.

## Endpoint

```text
https://api.devnet.solana.com
```
